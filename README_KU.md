# SURVIVOR ULTRA - وەشانی تەواو 🎮

یاریی زانیاری و کوێز بە کوردی - تەواو و چالاککراو

## تایبەتمەندییەکان ✨

### ١. سیستەمی یاری 🎯
- **یاری خێرا**: یاری 10 پرسیار لە هەر کاتێک
- **یاری سەرەکی**: یاریە کاتژمێرییەکان بە خەڵات زیاتر
- **تایمەر**: 15 چرکە بۆ هەر پرسیارێک
- **سیستەمی خاڵدان**: خاڵ بەپێی کات و ڕیزی وەڵامە ڕاستەکان
- **پرسیارە جۆراوجۆرەکان**: لە Firestore بارکراون
- **ئەنجامەکان**: پیشاندانی خاڵەکان و تۆکنە بەدەستهێنراوەکان

### ٢. سیستەمی فرۆشگا 🛒
- **50-50**: سڕینەوەی دوو وەڵامی هەڵە
- **تێپەڕاندن**: تێپەڕاندنی پرسیارێک
- **کاتی زیادە**: +10 چرکە کات
- **دوو هێندە**: دوو هێندە خاڵ بۆ پرسیاری داهاتوو
- **مێژووی کڕین**: لیستی هەموو کڕینەکان
- **پاداشتی ڕۆژانە**: تۆکنی خۆڕایی هەموو ڕۆژێک

### ٣. پەنەلی ئەدمین 👨‍💼
- **ئامارەکان**: ژمارەی یاریزانەکان، پرسیارەکان، یاریەکان
- **بەڕێوەبردنی پرسیارەکان**: 
  - زیادکردن
  - دەستکاریکردن
  - سڕینەوە
  - چالاک/ناچالاککردن
- **بەڕێوەبردنی یاریزانەکان**:
  - بینینی لیست
  - بینینی وردەکاری
  - سڕینەوە
- **هەناردەکردنی داتا**: CSV export بۆ یاریزانەکان و پرسیارەکان

### ٤. لیدەربۆرد (پلەبەندی) 🏆
- **بەرزترین 100**: لیستی باشترین یاریزانەکان
- **فیلتەرکردن**: هەموو کاتەکان، ئەم هەفتەیە، ئەم مانگە
- **گەڕان**: گەڕان بە ناوی یاریزان
- **پلەی خۆت**: پیشاندانی پلەی تۆ
- **مێدالیا**: 🥇🥈🥉 بۆ یەکەم سێ یاریزان

### ٥. سیستەمی دەسکەوت (Achievements) 🎖️
- **دەسکەوتی یاری**: 
  - یەکەم یاری، 10 یاری، 50 یاری، 100 یاری
- **دەسکەوتی خاڵ**: 
  - 100 خاڵ، 1000 خاڵ، 10000 خاڵ
- **دەسکەوتی تۆکن**: 
  - 100 تۆکن، 1000 تۆکن
- **دەسکەوتی بردنەوە**:
  - یەکەم بردنەوە، 10 بردنەوە
- **دەسکەوتی ئاست**:
  - ئاستی 5، ئاستی 10
- **دەسکەوتی تایبەت**:
  - یاریی تەواو (هەموو پرسیارەکان ڕاست)
  - خێرا وەک با (یاری < 30 چرکە)
  - گەڕانەوەی شاهانە (5 یاری لە ڕیزدا)
- **خەڵاتەکان**: تۆکنی خۆڕایی بۆ هەر دەسکەوتێک
- **پێشکەوتن**: پیشاندانی پێشکەوتنت بۆ هەر دەسکەوتێک

### ٦. پڕۆفایل 👤
- **زانیاری کەسی**: 
  - ناو، ئیمەیڵ، ئاستار
- **ئامارەکان**:

## چۆنیەتی دامەزراندن 🚀

### ١. پێداویستییەکان
- Web server (XAMPP, WAMP, Live Server)
- یان تەنها بکەرەوە لە browser

### ٢. ڕێکخستنی Firebase

لە `js/config.js` فایل:

```javascript
const firebaseConfig = {
    apiKey: "YOUR_API_KEY",
    authDomain: "YOUR_PROJECT.firebaseapp.com",
    projectId: "YOUR_PROJECT_ID",
    storageBucket: "YOUR_PROJECT.appspot.com",
    messagingSenderId: "YOUR_SENDER_ID",
    appId: "YOUR_APP_ID"
};
```

### ٣. ڕێکخستنی Firestore Rules

```javascript
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{userId} {
      allow read: if request.auth != null;
      allow write: if request.auth != null && request.auth.uid == userId;
    }
    
    match /questions/{questionId} {
      allow read: if request.auth != null;
      allow write: if request.auth != null && 
                      get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role == 'admin';
    }
    
    match /leaderboard/{userId} {
      allow read: if request.auth != null;
      allow write: if request.auth != null && request.auth.uid == userId;
    }
    
    match /purchases/{purchaseId} {
      allow read: if request.auth != null && 
                     resource.data.userId == request.auth.uid;
      allow create: if request.auth != null && 
                       request.resource.data.userId == request.auth.uid;
    }
  }
}
```

## چۆنیەتی زیادکردنی پرسیار ❓

### لە پەنەلی ئەدمین:
1. بچۆ بۆ بەشی "ئەدمین"
2. کلیک لە "زیادکردنی پرسیار"
3. پڕبکەرەوە:
   - پرسیارەکە
   - 4 وەڵام
   - وەڵامی ڕاست
   - پۆلێن (جوگرافیا، زانست، ...)
   - ئاستی سەختی (ئاسان، مامناوەند، قورس)
4. کلیک لە "زیادکردن"

### بە دەست لە Firestore:
```javascript
{
  question: "پرسیارەکە؟",
  answers: ["وەڵام 1", "وەڵام 2", "وەڵام 3", "وەڵام 4"],
  correct: 0,  // ئیندێکسی وەڵامی ڕاست (0-3)
  category: "جوگرافیا",
  difficulty: "easy",  // easy, medium, hard
  active: true,
  createdAt: serverTimestamp()
}
```

## چۆنیەتی تێستکردن 🧪

### ١. دروستکردنی هەژمار
- بچۆرە index.html
- کلیک لە "دروستکردنی هەژمار"
- ناو، ئیمەیڵ، پاسۆرد بنووسە

- بچۆ بۆ فرۆشگا

- لە یاری بەکاریان بهێنە

- چوونەژوورەوە بکە
