# Quick Start Guide - ڕێنمایی خێرا 🚀

## هەنگاوەکان بۆ دەستپێکردن

### ١. کردنەوەی پڕۆژەکە
```bash
# فایلی zip دابەزێنە و extract بکە
# دواتر index.html بکەرەوە لە browser
```

### ٢. دروستکردنی هەژمار
1. کلیک لە "دروستکردنی هەژمار"
2. ناو، ئیمەیڵ، پاسۆرد بنووسە
3. کلیک لە "دروستکردن"

✅ ئێستا دەتوانیت یاری بکەیت!

### ٣. دەستپێکردنی یەکەم یاری
1. لە پەڕەی سەرەکی
2. کلیک لە "یاری خێرا"
3. وەڵامی 10 پرسیار بدەرەوە
4. خاڵ و تۆکن بەدەست بهێنە!

### ٤. کڕینی توانایەکان
1. بچۆ بۆ "کۆگا" (فرۆشگا)
2. توانایەک هەڵبژێرە (50-50، تێپەڕاندن، ...)
3. کلیک لە "کڕین"
4. لە یاری داهاتوودا بەکاریان بهێنە!

### ٥. بینینی پلەبەندی
1. کلیک لە "پلەبەندی" لە navigation
2. ببینە لە کوێ دایت!
3. هەوڵبدە بچیتە Top 3! 🏆

---

## تێبینیە گرنگەکان ⚠️

### بۆ ئەدمینەکان
ئەگەر دەتەوێت ببیتە ئەدمین:

1. بچۆرە `js/config.js`
2. ئیمەیڵەکەت زیاد بکە:
```javascript
const ADMIN_EMAILS = [
    'your-email@gmail.com',  // ئەمە بگۆڕە
];
```
3. دووبارە refresh بکە
4. ئێستا دەتوانیت بچیتە پەنەلی ئەدمین!

### کێشەکانی باو

**کێشە:** بەشەکان پیشان نادەن
- **چارەسەر:** Console-ی Browser بکەرەوە (F12) و بگەڕێ بۆ هەڵە

**کێشە:** توانا کار ناکات
- **چارەسەر:** دڵنیابە تۆکنت پێویستە

**کێشە:** پەنەلی ئەدمین نییە
- **چارەسەر:** ئیمەیڵەکەت لە config.js زیاد بکە
